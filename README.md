# AI 熟議支援システム (AI Deliberation Support System)

生成AI（Google Gemini）を活用し、オンラインでの熟議（Deliberation）を活性化・支援するチャットプラットフォームです。
5W1H形式の提案作成支援や、AIによるファシリテーション機能を搭載しています。

## 機能
* **リアルタイムチャット:** WebSocketを使用したスムーズな対話
* **AIファシリテーター:** 議論の停滞時にAIが介入し、進行をサポート
* **5W1H提案フォーム:** 議論を具体的な政策提言に落とし込むための入力支援
* **リアルタイム要約:** 議論の内容をAIが分析し、構造化された議事録を生成
* **共有ノート:** 参加者全員で編集可能なリアルタイムエディタ
